# Goal Forge

## What is it?
A website that tracks everyday goals for a final big goal.


## Example:
Big goals can be
- Make the NBA
- Make the UFC

Small goals set inside that big goal can be
- Dribble every day for 20 minutes
- Hit the heavy bag for 20 minutes 

# Features
There is a + button to add a big goal, you can add multiple different big goals, within these big goals you can add small goals that you can check off which will be reset daily however there is an option for those small goals to be there everyday however it must be checked off daily, you can also manually add small goals on the current day you are on. 
When the big goal is reached, you can mark it as completed.
